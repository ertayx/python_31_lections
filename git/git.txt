локальный репозиторий - папка .git внутри нашего ноута
удаленный репозиторий - папка на сервере (которую мы создали в GitHub)

git init - создает git репозиторий из текущей папки

git add . - добавляет файлы и папки для отслеживания (эту команду пишем каждый раз перед командой commit)
git commit -m 'сообщения' - создает новую версию (сохраняет изменения), если были изменения

git status - показывает состояние файлов
git branch - показывает все ветки

git remote add origin {ssh|https} - создает связь между локальным и удаленным репозиторием
